# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

Customer Shopping Behavior Analysis is an end-to-end Data Analytics project developed using **Excel, MySQL, Python, and Power BI**. The project analyzes customer purchasing behavior to uncover meaningful business insights and presents them through an interactive dashboard.

---

## 🎯 Project Objectives

- Clean and preprocess customer shopping data.
- Perform SQL-based business analysis.
- Conduct Exploratory Data Analysis (EDA) using Python.
- Create interactive Power BI dashboards.
- Generate actionable business insights.

---

- Microsoft Excel
- MySQL
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

The dataset contains customer shopping information, including:

- Item Purchased

- Location
- Size
- Color
- Season
- Review Rating
- Subscription Status
- Shipping Type
- Discount Applied
- Promo Code Used
- Previous Purchases
- Payment Method
- Frequency of Purchases
